# dockerizedproxy

This is dockerized version of the [go-tcp-proxy](https://github.com/jpillora/go-tcp-proxy) project from [Jaime Pillora](dev@jpillora.com)

## Get it and start it
```sh
  git clone https://github.com/theskyinflames/dockerizedproxy.git
  cd dockerizedproxy
  make build
  make start
```
